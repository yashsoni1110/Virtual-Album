# Virtual Album — A Journey Through The Lens

A premium, high-performance web-based photography portfolio and virtual album built with **React**, **Framer Motion**, and **Tailwind CSS**. This project features a fluid masonry layout, smooth transitions, and an integrated media library.

## ✨ Features

- **🎨 Modern Aesthetics**: A clean, minimalist design focusing on visual storytelling with a sleek dark/light interface.
- **🖼️ Masonry Gallery**: Dynamically organized image grids that adapt to different screen sizes.
- **📂 Media Library**: Organized ceremonial archives with album-based navigation.
- **🔍 Premium Lightbox**: Immersive full-screen image viewing with smooth transitions.
- **🚀 Optimized Performance**: Powered by **Vite** and **ImageKit.io** for lightning-fast image delivery and lazy loading.
- **🎭 Smooth Animations**: Powered by **Framer Motion** for micro-interactions, page transitions, and scroll progress tracking.
- **📱 Fully Responsive**: Seamless experience across mobile, tablet, and desktop devices.

## 🛠️ Tech Stack

- **Frontend**: [React.js](https://reactjs.org/) (Vite)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Image Hosting**: [ImageKit.io](https://imagekit.io/)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yashsoni1110/Virtual-Album.git
   cd web-album
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
